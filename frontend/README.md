# Car Lock System Frontend

This is the frontend application for the Car Lock System, which provides a user interface to interact with the car locking features managed by the backend service.

## Project Structure

- **web/**: Contains the main web application files.
  - **src/**: Source files for the React application.
    - **App.tsx**: The main application component.
    - **components/**: Contains reusable components, including the LockControl component.
    - **api/**: Contains the API client for making requests to the backend.

## Getting Started

To get started with the frontend application, follow these steps:

1. **Install Dependencies**: Navigate to the `web` directory and run the following command to install the necessary dependencies:

   ```
   npm install
   ```

2. **Run the Application**: After installing the dependencies, you can start the development server with:

   ```
   npm start
   ```

3. **Access the Application**: Open your web browser and go to `http://localhost:3000` to access the application.

## Features

- User authentication and authorization.
- Control car locking features through a user-friendly interface.
- Real-time updates and interactions with the backend API.
